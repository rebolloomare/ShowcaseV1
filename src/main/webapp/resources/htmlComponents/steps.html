Plugin jQuery Steps

JQuery Steps es un componente de interfaz de usuario inteligente que te permite crear fácilmente interfaces de tipo asistente. Este complemento agrupa el contenido en secciones para obtener una vista de página más estructurada y ordenada. Además, es tan simple como 1-2-3 agregar complementos como jQuery Validation que puede evitar el cambio de pasos o la presentación.

Uso

  1. Descargar e incluir la librería jQuery en la sección de cabecera de su página. 
    				
	<script type="text/javascript" src="<c:url value='resources/jquery/plugins/steps/jquery.steps.min.js'/>"></script>
  2. Descargar e incluir el CSS en la sección de cabecera de su página. 
    				
	<link href="/resources/css/steps/jquery.steps.css'/" rel="stylesheet" type="text/css">

Configuración

Apariencia

Nombre de configuración Descripción Tipo Default
headerTag La etiqueta de encabezado se utiliza para buscar el texto del botón de paso dentro del área del asistente declarada. String h1
bodyTag La etiqueta de cuerpo se utiliza para encontrar el contenido del paso dentro del área del asistente declarada. String div
contentContainerTag La etiqueta del contenedor de contenido que se usará para envolver todo el contenido del paso. String div
actionContainerTag La etiqueta de contenedor de acción que se usará para envolver la navegación de paginación. String div
stepsContainerTag La etiqueta de contenedor de pasos que se utilizará para envolver la navegación de pasos. String div
cssClass La clase css que se agregará al contenedor externo. String wizard
stepsOrientation Determina si los pasos están orientados vertical u horizontalmente. Puede tener los valores:
horizontal ó 0
vertical ó 1		 String ó Integer horizontal ó 0

Plantillas

Nombre de configuración Descripción Tipo Default
titleTemplate La plantilla de título que se utilizará para crear un botón de paso. String <span class="number">#index#.</span> #title#
loadingTemplate La plantilla de carga que se utilizará para crear la animación de carga. String <span class="spinner"></span> #text#

Comportamiento

Nombre de configuración Descripción Tipo Default
autoFocus Establece el foco en la primera instancia del asistente para habilitar la navegación de las teclas desde el principio si es true. Boolean false
enableAllSteps Habilita todos los pasos desde el principio de true (todos los pasos se pueden hacer clic). Boolean false
enableKeyNavigation Habilite la navegación del teclado si es true (flecha izquierda y flecha derecha). Boolean true
enablePagination Habilite la paginación (siguiente, anterior y final) si es true. Boolean true
suppressPaginationOnFocus Suprime la paginación si se centra un campo de formulario. Boolean true
enableContentCache Habilitar caché para el contenido incrustado async o iframe. Boolean true
enableCancelButton Muestra el botón cancelar si está activado. Boolean false
enableFinishButton Muestra el botón Finalizar si está habilitado. Boolean true
showFinishButtonAlways Muestra el botón de finalizar siempre (en cada paso, al lado del siguiente botón) si es true. De lo contrario, el siguiente botón será reemplazado por el botón Finalizar si el último paso se activa. Boolean false
forceMoveForward Evita saltar a un paso anterior. Boolean false
saveState Guarda el estado actual (posición de paso) en una cookie. Llegando la próxima vez se activará el último paso activo. Boolean false
startIndex La posición de inicio (inicia en cero). Integer 0

Efectos de transición

Nombre de configuración Descripción Tipo Default
transitionEffect El efecto de animación que se utilizará para transiciones de pasos. Los valores pueden ser:
Sin animación de transición: none ó 0
Fade out/in en transición: fade ó 1
Deslice hacia abajo / hacia arriba la transiciónslide ó 2
Deslice la transición a la izquierdaslideLeft ó 3 		String ó Integer none ó 0
transitionEffectSpeed Velocidad de animación para transiciones de pasos (en milisegundos). Integer 200

Eventos

Nombre de configuración Descripción Tipo Default
onStepChanging Se ejecuta antes de que cambie el paso y se puede utilizar para evitar el cambio de paso al devolver false. Muy útil para la validación de formularios. Event function (event, currentIndex, newIndex) { return true; }
onStepChanged Se ejecuta después del paso han cambiado. Event function (event, currentIndex, priorIndex) { }
onCanceled Se ejecuta después de la cancelación. Event function (event) { }
onFinishing Se ejecuta antes de terminar y se puede utilizar para evitar la terminación de devolver falso. Muy útil para la validación de formularios. Event function (event, currentIndex) { return true; }
onFinished Se ejecuta después de la terminación. Event function (event, currentIndex) { }
onInit Se ejecuta cuando se inicia el asistente. Event function (event, currentIndex) { }
onContentLoaded Se ejecuta después del contenido asíncrono se cargan. Event function (event, currentIndex) { }

Etiquetas

Nombre de configuración Descripción Tipo Default
cancel Etiqueta para el botón Cancelar. String Cancel
current Esta etiqueta es importante por razones de accesibilidad. Indica qué paso está activado. String current step:
pagination Esta etiqueta es importante por razones de accesibilidad y describe el tipo de navegación. String Pagination
finish Etiqueta para el botón de finalizar. String Finish
next Etiqueta para el botón siguiente. String Next
previous Etiqueta para el botón anterior. String Previous
loading Etiqueta para la animación de carga. String Loading ...

Ejemplo de Configuración

			
var settings = {
    /* Appearance */
    headerTag: "h1",
    bodyTag: "div",
    contentContainerTag: "div",
    actionContainerTag: "div",
    stepsContainerTag: "div",
    cssClass: "wizard",
    stepsOrientation: $.fn.steps.stepsOrientation.horizontal,

    /* Templates */
    titleTemplate: '<span class="number">#index#.</span> #title#',
    loadingTemplate: '<span class="spinner"></span> #text#',

    /* Behaviour */
    autoFocus: false,
    enableAllSteps: false,
    enableKeyNavigation: true,
    enablePagination: true,
    suppressPaginationOnFocus: true,
    enableContentCache: true,
    enableCancelButton: true,
    enableFinishButton: true,
    preloadContent: false,
    showFinishButtonAlways: false,
    forceMoveForward: false,
    saveState: false,
    startIndex: 0,

    /* Transition Effects */
    transitionEffect: $.fn.steps.transitionEffect.none,
    transitionEffectSpeed: 200,

    /* Events */
    onStepChanging: function (event, currentIndex, newIndex) { return true; },
    onStepChanged: function (event, currentIndex, priorIndex) { }}, 
    onCanceled: function (event) { },
    onFinishing: function (event, currentIndex) { return true; }, 
    onFinished: function (event, currentIndex) { },

    /* Labels */
    labels: {
        cancel: "Cancel",
        current: "current step:",
        pagination: "Pagination",
        finish: "Finish",
        next: "Next",
        previous: "Previous",
        loading: "Loading ..."
    }
};
