Tabs

Las fichas se utilizan generalmente para romper el contenido en múltiples secciones que se pueden intercambiar para ahorrar espacio.
Las Tabs están contenidas en una lista ordenada <ol> o no ordenada <ul> . Cada "título" pestaña debe estar dentro de un elemento de la lista <li> y envuelto por un ancla <a> con un atributo "href".
Cada panel o pestaña puede ser cualquier elemento válido, teniendo un identificador que corresponda con el hash del anclaje.


Forma de Uso

Las Tabs son creadas con la clase <ul class="nav nav-tabs"> . Note que la clase .nav-tabs requiere la clase padre .nav base .
Tip: se recomienda marcar la pestaña activa por defecto con <li class="active"> .

Clases

Las clases usadas por el plugin tabs son para crear la navegación, agrupar contenido, hacer las pestañas dinámicas y responsivas en pantallas más pequeñas (recomendado para móviles).
A continuación se presentan las clases usadas:

Clase Descripción
.nav nav-tabs Crea los tabs de navegación.
.nav-justified Hace los tabs de navegación de la medida del contenedor "padre". En pantallas pequeñas, los tabs se apilan.
.tab-content Usado para crear tabs dinámicos. Se usa en conjunto con la clase .tab-pane y el atributo data-toggle="tab".
.tab-pane Usado para crear tabs dinámicos. Se usa en conjunto con la clase .tab-content y el atributo data-toggle="tab".

El atributo data-toggle="tab" se coloca en el elemento de referencia a, es decir, cuando se crean los tabs de navegación, en el listado <ul> .
La clase .tab-pane se coloca dentro de la clase contenedora .tab-content .
La clase .well se usa para darle un contorno al panel.

Ver el código de ejemplo en la pestaña Demo.

Desactivar Pestaña

Para desactivar una pestaña de navegación, se debe agregar la clases .disabled para establecer el enlace en color gris y agregar el efecto "desactivado".
Nota: Esta clase sólo cambiará la apariencia de <a> , no sus funciones. Se debe usar una función JavaScript personalizada para desactivar los enlaces.

Métodos

Método Descripción Ejemplo
show()
Muestra la pestaña o tab.
Este método no acepta ningún argumento.
 
<script>
$(document).ready(function(){
   $(".nav-tabs a").click(function(){
      $(this).tab('show');
   });
});
</script>

Eventos

Evento Descripción
show.bs.tab
Ocurre cuando la pestaña está a punto de ser mostrada.
shown.bs.tab
Ocurre cuando la pestaña está totalmente mostrada (después de haber completado transiciones CSS).
hide.bs.tab
Ocurre cuando la pestaña está a punto de ser ocultada.
hidden.bs.tab
Ocurre cuando la pestaña está totalmente ocultada (después de haber completado transiciones CSS).
