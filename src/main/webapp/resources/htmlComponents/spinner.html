Spinner con Métodos


Widget de Spinner

El Spinner, o Widget Numérico paso a paso, es perfecto para el manejo de todo tipo de entrada numérica. Permite a los usuarios escribir un valor directamente, o modificar un valor existente haciendo girar con el teclado, el mouse o la rueda del mouse. Cuando se combina con Globalize , incluso se puede hacer girar las monedas y las fechas en una variedad de lugares.
Spinner contiene una entrada de texto con dos botones para aumentar y disminuir el valor actual, se añaden eventos clave para que el mismo incremento y decremento se pueden hacer con el teclado.

Manejo del Teclado
  • UP: Incrementa el valor en un paso.
  • DOWN: Decrementa el valor en un paso.
  • PAGE UP: Incrementa el valor en una página.
  • PAGE DOWN: Decrementa el valor en una página.
Notas adicionales:
  • Este widget requiere funcionalidad CSS, de lo contrario no funcionará. Si se construye un tema personalizado, utilice el archivo CSS específico del widget como punto de partida.
  • Este widget puede manipular el valor de su elemento mediante programacián, por lo tanto, un evento nativo change no puede ser ejecutado cuando cambia el valor del elemento.
  • La creación de una ruleta en un <input type="number"> no se admite debido a un conflicto con la interfaz de usuario nativa.

Tipos de Spinner

Se puede dar formato al valor del Spinner, esto es, usando las opciones del widget, a continuación se muestra dos tipos:
  • Monetario
  • Hora
Se presentará la función en JavaScript y la invocación en Html.

Spinner: Monetario

El formato del valor del Spinner es en Decimal, así como se agrega la etiqueta de la moneda, de acuerdo a la selección del Globalize.
Función JavaScript
$(function() {
	$("#currency").on("change", function() {
		$("#spinner").spinner("option", "culture", $(this).val());
	});

	$("#spinner").spinner({
		min : 5,
		max : 2500,
		step : 25,
		start : 1000,
		numberFormat : "C"
	});
});
Invocación HTML
<p>
  <label for="spinner">Amount to donate:</label>
  <input id="spinner" name="spinner" value="5">
</p> 
<p>
  <label for="currency">Currency to donate</label>
  <select id="currency" name="currency">
    <option value="en-US">US $</option>
    <option value="de-DE">EUR €</option>
    <option value="ja-JP">YEN ¥</option>
  </select>
</p>

Spinner: Hora

El formato del valor del Spinner es formateado para hora (time), así como se agrega la etiqueta de tiempo: PM o AM, de acuerdo a la selección del Globalize.
Función JavaScript
$(function() {
	$.widget("ui.timespinner", $.ui.spinner, {
		options : {
			// seconds
			step : 60 * 1000,
			// hours
			page : 60
		},
		_parse : function(value) {
			if (typeof value === "string") {
				// already a timestamp
				if (Number(value) == value) {
					return Number(value);
				}
				return +Globalize.parseDate(value);
			}
			return value;
		},
		_format : function(value) {
			return Globalize.format(new Date(value), "t");
		}
	});
	$("#spinner").timespinner();
	$("#culture").on("change", function() {
		var current = $("#spinner").timespinner("value");
		Globalize.culture($(this).val());
		$("#spinner").timespinner("value", current);
	});
});
Invocación HTML
<p>
  <label for="spinner">Time spinner:</label>
  <input id="spinner" name="spinner" value="08:30 PM">
</p>
<p>
  <label for="culture">Select a culture to use for formatting:</label>
  <select id="culture">
    <option value="en-EN" selected="selected">English</option>
    <option value="de-DE">German</option>
  </select>
</p>

Opciones

culture

Establece el tipo de cultura a usar para analizar y dar formato al valor. Si es null, se usa la cultura establecida por defecto en el plugin Globalize.
Tipos: String
Default: null
Ejemplos de Código
Inicializando el Spinner con la opción culture especificada:
$( ".selector" ).spinner({
culture: "fr"
});
Obtener o establecer la opción culture, después de inicializar:
// Getter
var culture = $( ".selector" ).spinner( "option", "culture" );
// Setter
$( ".selector" ).spinner( "option", "culture", "fr" );

disabled

Desabilita el Spinner al establecer el valor true.
Tipos: Boolean
Default: false
Ejemplos de Código
Inicializando el Spinner con la opción disabled especificada:
$( ".selector" ).spinner({
disabled: true
});
Obtener o establecer la opción disabled, después de inicializar:
// Getter
var disabled = $( ".selector" ).spinner( "option", "disabled" );
// Setter
$( ".selector" ).spinner( "option", "disabled", true );

icons

Los iconos a utilizar para los botones, iconos proporcionados por el framework de jQuery UI CSS.
  • up (string, default: "ui-icon-triangle-1-n")
  • down (string, default: "ui-icon-triangle-1-s")
Tipos: Object
Default: { down: "ui-icon-triangle-1-s", up: "ui-icon-triangle-1-n" }
Ejemplos de Código
Inicializando el Spinner con la opción icons especificada:
$( ".selector" ).spinner({
icons: { down: "custom-down-icon", up: "custom-up-icon" }
});
Obtener o establecer la opción icons, después de inicializar:
// Getter
var icons = $( ".selector" ).spinner( "option", "icons" );
// Setter
$( ".selector" ).spinner( "option", "icons", { down: "custom-down-icon", up: "custom-up-icon" } );

incremental

Controla el número de pasos dados cuando se mantiene pulsado un botón de giro.
Tipos: Boolean o Function(Integer count)
Default: true

Múltiples valores soportados:
Boolean: Cuando se define como true, el paso tipo "delta" se incrementará cuando se gire sin cesar. Cuando se establece en false, todos los pasos son iguales (como se define por la opción step).
Function: Recibe un parámetro: el número de giros que se han producido. Debe devolver el número de pasos que deben darse para el giro actual.
Ejemplos de Código
Inicializando el Spinner con la opción incremental especificada:
$( ".selector" ).spinner({
incremental: false
});
Obtener o establecer la opción incremental, después de inicializar:
// Getter
var incremental = $( ".selector" ).spinner( "option", "incremental" );
// Setter
$( ".selector" ).spinner( "option", "incremental", false );

max

El valor máximo permitido, este atributo max del elemento se utiliza si existe ya que la opción no se establece explícitamente. Si es null, no hay un valor máximo límite.
Tipos: Number o String
Default: null

Múltiples valores soportados:
Number: El valor máximo permitido.
String: Si se incluye Globalize, la opción max se puede pasar como una cadena que será analizada en función de las opciones numberFormat y culture; de lo contrario se caerá de nuevo al método nativo parseFloat().
Ejemplos de Código
Inicializando el Spinner con la opción max especificada:
$( ".selector" ).spinner({
max: 50
});
Obtener o establecer la opción max, después de inicializar:
// Getter
var max = $( ".selector" ).spinner( "option", "max" );
// Setter
$( ".selector" ).spinner( "option", "max", 50 );

min

El valor mínimo permitido, este atributo min del elemento se utiliza si existe ya que la opción no se establece explícitamente. Si es null, no hay un valor mínimo límite.
Tipos: Number o String
Default: null

Múltiples valores soportados:
Number: El valor mínimo permitido.
String: Si se incluye Globalize, la opción min se puede pasar como una cadena que será analizada en función de las opciones numberFormat y culture; de lo contrario se caerá de nuevo al método nativo parseFloat().
Ejemplos de Código

// Setter

numberFormat

Formatos de números basados en Globalize. Las más comunes son "n" para un número decimal y un "C" para un valor de moneda. Consulte la opción culture.
Tipos: String
Default: null
Ejemplos de Código
Inicializando el Spinner con la opción numberFormat especificada:

// Setter

page

El número de pasos que se tomarán cuando el avance sea vía los métodos pageUp / pageDown.
Tipos: Number
Default: 10
Ejemplos de Código

El tamaño del paso que se toma en el avance por medio de los botones o por los métodos stepUp / stepDown.
Tipos: Number o String
Default: 1

Múltiples valores soportados:
Number: El valor del tamaño del paso.
String: Si se incluye Globalize, la opción step se puede pasar como una cadena que será analizada en función de las opciones numberFormat y culture; de lo contrario se caerá de nuevo al método nativo parseFloat().
Ejemplos de Código

Métodos

Método Descripción Ejemplo
destroy()
 Elimina la funcionalidad por completo. Esto devolverá el elemento de nuevo a su estado previo de inicio.
Este método no acepta ningún argumento.
$( ".selector" ).spinner( "destroy" );
disable()
 Inhabilita el Spinner.
Este método no acepta ningún argumento.
$( ".selector" ).spinner( "disable" );
enable()
 Habilita el Spinner.
Este método no acepta ningún argumento.
$( ".selector" ).spinner( "enable" );
instance()
 Recupera el objeto de instancia del control deslizante. Si el elemento no tiene una instancia asociada,
devuelve undefined. A diferencia de otros métodos de controles, instance() es seguro
llamar en cualquier elemento después de que el plugin de Spinner se ha cargado.
Este método no acepta ningún argumento.
$( ".selector" ).spinner( "instance" );
isValid()
 Devuelve el valor del Spinner si es válido de acuerdo a los valores de la opciones
min, max y step.
Este método no acepta ningún argumento.
$( ".selector" ).spinner( "isValid" );
option(optionName)
 Obtiene el valor especificado en optionName.
Para las opciones que tienen objetos como su valor, se puede obtener el valor de una clave específica
mediante el uso de la notación de punto. Por ejemplo, "foo.bar" obtendría el valor de
la propiedad de barra en la opción foo.
optionName: el nombre de la opción a obtener.
Type: String
var isDisabled = $( ".selector" ).spinner( "option", "disabled" );
option()
 Recupera un objeto que contiene los pares de atributo/valor que representan las actuales opciones del Spinner.
Este método no acepta ningún argumento.
var options = $( ".selector" ).spinner( "option" );
option(optionName, value)
 Establece el valor de la opción del Spinner especificada en optionName.
Para las opciones que tienen objetos como su valor, se puede establecer el valor de una propiedad
específica mediante el uso de la notación de punto. Por ejemplo, "foo.bar"
actualiza el valor de la propiedad de barra en la opción foo.
optionName: el nombre de la opción a obtener.
Type: String
value: el valor a establecer en la opción.
Type: Object
$( ".selector" ).spinner( "option", "disabled", true );
option(options)
 Establece uno o más opciones al Spinner.
options: un mapa de pares de opciones-valores a establecer.
Type: Object
$( ".selector" ).spinner( "option", { disabled: true } );
pageDown([pages])
 Reduce el valor por el número especificado de páginas, como se define por la opción page. Sin el
parámetro, una sola página se decrementa. Si el valor resultante está por encima de max,
o debajo de min, o resulta en una falta de coincidencia, el valor se ajusta al valor válido más
cercano.
Invocando el método pageDown causará que los eventos start, spin y stop sean activados.
pages: número de páginas a decrementar, por default es 1.
Type: Number
$( ".selector" ).spinner( "pageDown" );
pageUp([pages])
 Incrementa el valor por el número especificado de páginas, como se define por la opción page. Sin el
parámetro, una sola página se incrementa. Si el valor resultante está por encima de max,
o debajo de min, o resulta en una falta de coincidencia, el valor se ajusta al valor válido más
cercano.
Invocando el método pageUp causará que los eventos start, spin y stop sean activados.
pages: número de páginas a incrementa, por default es 1.
Type: Number
$( ".selector" ).spinner( "pageUp" );
stepDown([steps])
 Reduce el valor de pasos por el número especificado, como se define por la opción steps. Sin el
parámetro, un solo paso se decrementa. Si el valor resultante está por encima de max,
o debajo de min, o resulta en una falta de coincidencia, el valor se ajusta al valor válido más
cercano.
Invocando el método stepDown causará que los eventos start, spin y stop sean activados.
steps: número de pasos a decrementar, por default es 1.
Type: Number
$( ".selector" ).spinner( "stepDown" );
stepUp([steps])
 Aumenta el valor de pasos por el número especificado, como se define por la opción steps. Sin el
parámetro, un solo paso se incrementa. Si el valor resultante está por encima de max,
o debajo de min, o resulta en una falta de coincidencia, el valor se ajusta al valor válido más
cercano.
Invocando el método stepUp causará que los eventos start, spin y stop sean activados.
steps: número de pasos a incrementar, por default es 1.
Type: Number
$( ".selector" ).spinner( "stepUp" );
value()
 Obtiene el valor del Spinner.
Este método no acepta ningún argumento.
var selection = $( ".selector" ).spinner( "value" );
value(value)
 Establece el valor al Spinner.
value: el valor a establecer.
Type: Number
$( ".selector" ).spinner( "value", 55 );
widget()
 Regresa un objeto jQuery conteniendo el Spinner.
Este método no acepta ningún argumento.
var widget = $( ".selector" ).spinner( "widget" );

Eventos

change(event, ui)

Se activa cuando el valor del Spinner ha cambiado y la entrada ha perdido el foco.

  • event

    Tipo: Event

  • ui

    El objeto ui es vacío pero es incluido para consistencia con los otros eventos.

Ejemplo. Inicializando el Spinner con la llamada al evento change:
$( ".selector" ).spinner({
change: function( event, ui ) {}
});

create(event, ui)

Se activa cuando el Spinner es creado.

  • event

    Tipo: Event

  • ui

    El objeto ui es vacío pero es incluido para consistencia con los otros eventos.

Ejemplo. Inicializando el Spinner con la llamada al evento create:
$( ".selector" ).spinner({
create: function( event, ui ) {}
});

spin(event, ui)

Se activa durante el incremento/decremento del valor en el Spinner (para determinar la dirección de giro compara el valor actual con el del ui.value). Se puede cancelar, evitando que el valor se actualice.

  • event

    Tipo: Event

  • ui

    Tipo: Object

    • value

      Tipo: Number
      El nuevo valor que se determine, a menos que el evento se cancele.

Ejemplo. Inicializando el Spinner con la llamada al evento spin:
$( ".selector" ).spinner({
spin: function( event, ui ) {}
});

start(event, ui)

Se activa antes de dar una vuelta. Se puede cancelar, impidiendo el giro que se produzca.

  • event

    Tipo: Event

  • ui

    El objeto ui es vacío pero es incluido para consistencia con los otros eventos.

Ejemplo. Inicializando el Spinner con la llamada al evento start:
$( ".selector" ).spinner({
start: function( event, ui ) {}
});

stop(event, ui)

Se activa después de una vuelta.

  • event

    Tipo: Event

  • ui

    El objeto ui es vacío pero es incluido para consistencia con los otros eventos.

Ejemplo. Inicializando el Spinner con la llamada al evento stop:
$( ".selector" ).spinner({
stop: function( event, ui ) {}
});
