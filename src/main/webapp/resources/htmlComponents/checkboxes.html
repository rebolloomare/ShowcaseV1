Checkbox

Un Checkbox es usado si usted quiere seleccionar algunas de las opciones desde una lista de opciones predeterminada.

Para usarlos hay que rodear la etiqueta <input type="checkbox" ... entre etiqueta <div class='checkbox'> con la clase checkbox, ejemplo:


	
		<div class="checkbox">
			<label><input type="checkbox" value="">Opción X</label>
		</div>

Si usted desea deshabilitar un checkbox agregue el atributo disabled a la etiqueta <div> ., ejemplo: 


	
		<div class="checkbox disabled">
			<label><input type="checkbox" value="">Opción X</label>
		</div>
