Plugin PickList

Es un plugin de selección de lista potente y fácil. Convierte un cuadro de selección múltiple estándar en una lista de selección que permite intercambiar elementos de lado a lado, uno o varios, con facilidad y rapidez.


Uso

  1. Descargar e incluir la librería jQuery en la sección de cabecera de su página. 
    				
	<script type="text/javascript" src="<c:url value='resources/jquery/plugins/picklist/pickList.js'/>"></script>
  2. Descargar e incluir el CSS en la sección de cabecera de su página. 
    				
	<link href="resources/css/picklist/pickList.css'/" rel="stylesheet" type="text/css">
  3. Generar un arreglo de objetos para la lista de la izquierda. indicando un id y un valor text para cada elemento de la lista 
    				
	var val = {
		01: {id: 556, text: 'Azteca'},
		02: {id: 552, text: 'Banamex'},
		03: {id: 568, text: 'Coppel'},
		04: {id: 550, text: 'Inbursa'},
		05: {id: 562, text: 'Invercap'},
		06: {id: 564, text: 'Metlife'},
		07: {id: 578, text: 'Pensión ISSSTE'},
		08: {id: 538, text: 'Principal'},
		09: {id: 534, text: 'Profuturo GNP'},
		10: {id: 544, text: 'Sura'},
		11: {id: 530, text: 'XXI Banorte'}
	};
  4. Personalizar los botónes para intercambio de elementos entre las listas. (Paso opcional) 
    				
	var etiquetas = {
		add: 'Agrega Afore >',
		addAll: 'Agregar Todas >>',
		remove: '< Remover Afore',
		removeAll: 'Remover Todas >>'
	};

Propiedades

Etiquetas

Nombre Descripción Tipo Default
add Etiqueta para mover a la derecha un elemento de la lista String Agregar >
addAll Etiqueta para mover a la derecha todos los elementos de la lista String Agregar Todos >>
remove Etiqueta para mover a la izquireda un elemento de la lista String < Remover
removeAll Etiqueta para mover a la izquierda todos los elementos de la lista String << Remover Todos

Métodos

Nombre Descripción
getValues Guarda en un arreglo de objetos los elementos en la lista de la derecha (seleccionados). Los objetos se identifican por id y text
