Plugin Modal

El plugin Modal es una ventana caja/popup que es desplegada sobre la página actual.

Nota: El plugin puede ser incluido individualmente (usando el archivo individual Bootstrap "modal.js"), o todo de una vez (usando "bootstrap.js" o "bootstrap.min.js").

Explicación del ejemplo:

Para lanzar la ventana modal, necesita usar un botón o un link.
Entonces incluir los dos atributos data-* attributes:

  • data-toggle="modal" abre una ventana modal
  • data-target="#myModal" apunta al id del modal
La parte "Modal":

La etiqueta padre <div> del modal debe tener un ID que el es mismo valor pasado en el atributo data-target usado para lanzar el modal ("myModal").
La clase.modal identifica el contenido del elemento <div> como un modal y pone el foco en él.
La clase .fade dá el efecto de transición el cual aparece o desaparece el modal. Remueva esta clase si quiere eliminar este efecto.
El atributo role="dialog" agrega la facilidad para personas que usan pantallas lectoras .
La clase .modal-dialog configura la propiedad ancho y el margen del modal.

El contenido del modal:

La etiqueta <div> con la clase class="modal-content" estiliza el modal (bordes, color de fondo, etc.). Dentro del <div> se agrega el header, body, y el footer. La clase .modal-header es usada para definir el estilo para el header. El <button> dentro del header tiene el atributo data-dismiss="modal" el cual cierra el modal si hace click en él. La clase .close estiliza el botón "close", y la clase .modal-title estiliza el header con una propiedad line-height.
La clase .modal-body es usada para definir el estilo del cuerpo en el modal.
La clase .modal-footer es usada para definir el estilo del footer. Note que esta area esta alineada a la derecha por default.



Tamaño del Modal

Cambie el tamaño del modal agregando la clase .modal-sm para modales pequeños, la clase .modal-lg para modales grandes.
Agregar estas clases al <div> que tiene la clase .modal-dialog.
