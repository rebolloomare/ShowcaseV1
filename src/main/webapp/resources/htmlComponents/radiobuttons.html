Botones de radio

Los Radio buttons son usados si usted quiere limitar a solo una opción desde una lista preestablecida de opciones.

Para usarlos hay que rodear la etiqueta <input type="radio" ... entre etiqueta <div class='radio'> con la clase radio, ejemplo: 

		<div class="radio">
			<label><input type="radio" name="optradio" >Opción X</label>
		</div>

Botones de radio deshabilitados

Usar la clase disabled , ejemplo: 

		<div class="radio disabled">
			<label><input type="radio" name="optradio" >Opción X</label>
		</div>
