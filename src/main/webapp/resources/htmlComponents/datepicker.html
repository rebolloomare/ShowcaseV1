Datetimepicker

Datetimepicker es un componente que muestra un calendario nos ayuda a insertar la fecha seleccionada en un campo de texto, está desarrollado como un plugin de jQuery

Configurando Datetimepicker

Se requiere agregar las librerías CSS (bootstrap-datetimepicker.css) y Javascript de datetimepicker (bootstrap-datetimepicker.js y moment-with-locales.js) en el encabezado de la página HTML para poder trabajar con este plugin, ejemplo:

		
		<!DOCTYPE html>
		<html>
			<head>
			
				<!-- estilos datetimepicker -->
				<link  href="css/bootstrap-datetimepicker.css" rel="stylesheet" type="text/css" >
				<!-- scripts datetimepicker -->
				<script type="text/javascript" src="/js/moment-with-locales.js'/>"></script>
				<script type="text/javascript" src="js/bootstrap-datetimepicker.js'/>"></script>
				<!-- además se requiere el script de jQuery -->
				<script src="jquery.js" type="text/javascript"></script>
			....

Iniciando Datetimepicker configuración por default

Para iniciar el elemento que mostrará el botón para iniciar el calendario y recibirá la fecha al seleccionarla hay que usar la siguiente línea de código $('#id_input_selector').datetimepicker();, donde id_input_selector es el ID del elemento HTML que mostrara el calendario

El elemento HTML que mostrará el campo de texto con la fecha y el botón para mostrar el calendario se muestra con el siguiente código:

		<div class="form-group"> <!-- agrupador -->
			<div class='input-group date' id='id_input_selector'> <!-- id -->	
				<input type='text' class="form-control" /> <!-- campo de texto para la fecha 	-->
				<span class="input-group-addon"> <!-- muestra botón lanzador del calendario -->
					<!-- muestra el icono para el botón -->
					<span class="glyphicon glyphicon-calendar"></span> 
				</span>
			</div>
		</div>
