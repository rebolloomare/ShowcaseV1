Widget de Tooltip


El Tooltip plugin es una pequeña caja pop-up que aparece cuando el usuario mueve el puntero del mouse sobre un elemento.

Para crear un tooltip, agregue el atributo data-toggle="tooltip" a está propiedad. Use el atributo title para especificar el texto que va ser mostrado dentro del tooltip.

Iniciando tooltip:

Los tooltips se deben iniciar usando jQuery: seleccionado el elemento especificado y llamando al método tooltip() . El siguiente código va habilitar todos los tooltips en el documento: 

<script type="text/javascript">	
	$(document).ready(function() {
		$('[data-toggle="tooltip"]').tooltip();
	});		
</script>
Posiciones del Tooltip
Por defecto, un Tooltip se posiciona al arriba del elemento. Para cambiar dicha posición, es necesario usar el atributo data-placement para visualizar el Tooltip en diferentes lados del elementos, los cuales pueden ser:
  • Arriba
  • Abajo
  • Derecha
  • Izquierda

Opciones

Las opciones de un Tooltip pueden ser enviadas mediante atributos o por JavaScript.
Si se usan los atributos, se debe agregar el nombre de la opción al prefijo data-, como por ejemplo data-placement = "".

Se muestra la siguiente tabla de atributos:

Opción Tipo Valor Default Descripción
animation Boolean true Especifica si se agrega un efecto CSS de transición de desvanecimiento cuando se abre o se cierra el Tooltip.
  • true: se agrega el efecto.
  • false: no se agrega el efecto.
container String, Boolean false Agrega el Tooltip a un elemento específico. Ejemplo container: 'body'
delay Number, Object 0 Especifica el número de milisegundos que tomará el Tooltip en abrir o cerrar.
Para especificar un tiempo de delay para abrir y otro para cerrar el Tooltip, se debe usar la siguiente estructura:
delay: {show: 500, hide: 100} el cual tomará 500ms en abrir el Tooltip, pero solo 100ms para cerrarlo.
html Boolean false Especifica si se aceptan tags html en el Tooltip.
  • true: acepta tags HTML.
  • false: no se aceptan tags HTML.
Nota: el HTML debe ser insertado en el atributo title o usando la misma opción.
placement String "top" Especifica la posición del Tooltip. Posibles valores:
  • top: arriba
  • buttom: abajo
  • left: lado izquierdo
  • right: lado derecho
  • auto: El Navegador decide la posición, es decir, si el valor es "auto left" el Tooltip se visualizará a la izquierda mientras sea posible, de lo contrario,
    se mostrará a la derecha; si el valor es "auto bottom" el Tooltip se visualizará abajo mientras sea posible, de lo contrario se mostrará arriba.
selector String, Boolean false Agrega el Tooltip a un selector específico.
template String Se usa como base HTML para crear el Tooltip
El título del Tooltip debe ser inyectado dentro de .tooltip-inner
La clase .tooltip-arrow se convertirá en la flecha del Tooltip
El elemento más externo es quien contendrá la clase .tooltip
title String "" Especifica el texto que se muestra en el interior del Tooltip.
trigger String "hover focus" Especifica como el Tooltip será activado. Posibles valores:
  • "click": activa el Tooltip con un click.
  • "hover": activa el Tooltip con el puntero del mouse.
  • "focus": activa el Tooltip cuando tiene el foco.
  • "manual": activa el Tooltip manualmente.
Nota: para pasar múltiples valores, se deben separar mediante un espacio.

Métodos

Se muestra la siguiente tabla de métodos disponibles:

Método Descripción
.tooltip(options) Activa el Tooltip con una opción. Ver la tabla de Opciones con los valores válidos.
.tooltip("show") Muestra el Tooltip.
.tooltip("hide") Oculta el Tooltip
.tooltip("toggle") Alterna el Tooltip.
.tooltip("destroy") Oculta y destruye el Tooltip.

Eventos

Se muestra la siguiente tabla de eventos disponibles:

Evento Descripción
show.bs.tooltip Se produce cuando el Tooltip está a punto de ser mostrado.
shown.bs.tooltip Ocurre cuando el Tooltip está totalmente muestrado (después de que la transición CSS es completada).
hide.bs.tooltip Se produce cuando el Tooltip está a punto de ser ocultado.
hidden.bs.tooltip Ocurre cuando el Tooltip está totalmente ocultado (después de que la transición CSS es completada).
