Conceptos básicos de HTML y Javascript

CanvasJS

CanvasJS es una biblioteca fácil de utilizar HTML5 y Javascript de gráficos. Se ejecuta a través de dispositivos como el iPhone, iPad, Android, Windows Phone, Microsoft Surface, ordenadores de sobremesa, etc. Esto le permite crear cuadros de mando ricos que funcionan a través de dispositivos sin comprometer la funcionalidad o la mantenibilidad.

Requisitos del Navegador

Es compatible con todos los navegadores modernos. A partir de la versión 1.3, CanvasJS también es compatible con los navegadores más antiguos como IE6, IE7 y IE8.

Descripción del Gráfico

Un gráfico de líneas es un tipo de gráfico que muestra información como una serie de puntos de datos conectados por segmentos de línea recta. Cada punto de datos tiene x variable de la determinación de la posición en el eje y variable y horizontal la determinación de la posición del eje vertical. Los marcadores se desactivan automáticamente cuando hay gran número de puntos de datos. Puede anular este comportamiento estableciendo manualmente markerSize a un valor mayor a cero. No se puede combinar con: circulares, de barras, de barras apiladas.
