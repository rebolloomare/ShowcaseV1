Sistema Grid Bootstrap

El sistema de Grid de Bootstrap permite dividir la pantalla en múltiples filas y columnas que pueden ser usadas para crear diferentes tipos de layouts. Una vez que tenemos las filas y columnas que hayamos decidido, podemos incrustar el código HTML que va ir dentro para formar los componentes WEB requeridos como: formularios, menús, paneles, imágenes, tablas, etc...

Estructura básica de un grid

1.- Debemos usar un contenedor, la etiqueta <div> con alguna de las clases container o container-fluid, dentro del contenedor se coloca la fila o filas deseadas usando la clase row en combinación de un <div>:


	<div class="container"> <!-- contenedor -->
		<div class="row">	<!-- fila -->
		</div>
	</div>

La clase container: Crea un contenedor ocupando un ancho fijo dentro de la ventana del navegador.


La clase container-fluid: Crea un contenedor dinámico que se adecua usando el ancho completo de la ventana del navegador.


Se pueden usar infinidad de filas:


	<div class="container">
		
		<!-- varias filas -->
		
		<div class="row"></div> 
		<div class="row"></div>
		<div class="row"></div>
		<div class="row"></div>
		...
		
	</div>

2.- Por cada fila puede contener una o varias columnas que en total cada fila usar cada una de las 12 columnas individualmente, o puede agrupar columnas para crear columnas más anchas:

			
			<div class="row">
			 <div class="col-*-*"></div>		<!-- una fila con una columna -->
			</div>
			
			<!-- una fila con 3 columnas  -->
			<div class="row">
			  <div class="col-*-*"></div>
			  <div class="col-*-*"></div>
			  <div class="col-*-*"></div>
			</div>
			<div class="row">
			  ...
			</div>

Se usa la clase: col-*-* para indicar que es una columna, el primer * se sustituye por el tipo de pantalla el segundo * se sustituye por el ancho en columnas (1 a 12), ejemplos:

  • col-md-12: Ocupa ancho de 12 columnas
  • col-md-1: Ocupa ancho de 1 columna
  • col-md-6: Ocupa ancho de 6 columnas
  • col-md-3: Ocupa ancho de 3 columnas

Ancho de columnas en distintas pantallas

En Bootstrap puede especificar distintos anchos de columnas para distintos tipos de pantallas a la vez. Usando alguna de los siguientes cuatro tipos:

  • xs (para telefonos)
  • sm (para tablets)
  • md (para pantallas de computadoras)
  • lg (para pantallas de computadoras largas)

El siguiente código especifica una fila con 2 columnas, cada columna ocupara una fila y 6 columnas de ancho en pantallas medianas, en pantallas pequeñas cada columna estará en una fila usando todo el ancho de la pantalla (12 columnas):


		<div class="container">
			<div class="row">
				<div class="col-xs-12 col-sm-6">				
					<h4>Column 1</h4>
				</div>
				
				<div class="col-xs-12 col-sm-6">
					<h4>Column 2</h4>
				</div>
			</div>
		</div>
