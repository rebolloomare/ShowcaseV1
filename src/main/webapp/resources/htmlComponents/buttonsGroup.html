Botones agrupados:

Bootstrap permite agrupar series de botones juntos (en una sola línea) en un button group:

Use la etiqueta <div> con la clase .btn-group para crear un button group.



Tamaños para botones agrupados:

Se puede asignar el tamaño a todos los botones del grupo de una sola vez, usando la clase .btn-group-*.



Botones agrupados en vertical:

Bootstrap soporta botones en forma vertical en un Button Group, usando la clase .btn-group-vertical.




Justificación de grupo de botones:

Para expandir usando todo el ancho de la pantalla use la clase .btn-group-justified.



Mezclando grupo de botones y menus Dropdown:

Se puede anidar "Grupos de botones" para crear menús desplegables, ver el ejemplo en la pestaña demo.
