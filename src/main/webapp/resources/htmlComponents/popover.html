Widget de Popover

El plug-in Popover es similar a los "ToolTips". Se trata de un cuadro emergente que aparece cuando el usuario hace clic en un elemento.
La diferencia es que el Popover puede contener mucho más contenido.


Forma de Uso

Para crear un Popover basta agregar el atributo data-toggle="popover" a un elemento.
Use el atributo title para especificar el texto como título del Popover, y use el atributo data-content para especificar el texto que se mostrará dentro del cuerpo del Popover.

Nota: Es necesario inicializar mediante jQuery, haciendo la invocación del método popover() (ver la pestaña Código de este widget).

Posiciones del Popover
Por defecto, un Popover se posiciona al lado derecho del elemento. Para cambiar dicha posición, es necesario usar el atributo data-placement para visualizar el Popover en diferentes lados del elementos, los cuales pueden ser:
  • Arriba
  • Abajo
  • Derecha
  • Izquierda
Opciones de Cerrado del Popover
Por defecto, el Popover se cierra cuando se hace clic en el elemento de nuevo. Sin embargo, se puede utilizar el atributo data-trigger="focus" que cerrará la Popover al hacer clic fuera del elemento.

Si se desea que el Popover se visualice cuando se mueva el puntero del mouse sobre el elemento, y desaparezca cuando el puntero esté fuera del mismo; se debe usar el atributo data-trigger="hover". Esto creará un Popover tipo Flotante.

Atributos: data-*

Atributo Descripción
data-toggle="popover" Activa el Popover.
title Especifica el texto como título del Popover.
data-content Especifica el texto que se mostrará dentro del cuerpo del Popover.

Opciones

Las opciones de un Popover pueden ser enviadas mediante atributos o por JavaScript.
Si se usan los atributos, se debe agregar el nombre de la opción al prefijo data-, como por ejemplo data-placement = "".

Se muestra la siguiente tabla de atributos:

Opción Tipo Valor Default Descripción
animation Boolean true Especifica si se agrega un efecto CSS de transición de desvanecimiento cuando se abre o se cierra el Popover.
  • true: se agrega el efecto.
  • false: no se agrega el efecto.
container String, Boolean false Agrega el Popover a un elemento específico. Ejemplo container: 'body'
content String "" Especifica el texto que se muestra dentro del cuerpo del Popover.
delay Number, Object 0 Especifica el número de milisegundos que tomará el Popover en abrir o cerrar.
Para especificar un tiempo de delay para abrir y otro para cerrar el Popover, se debe usar la siguiente estructura:
delay: {show: 500, hide: 100} el cual tomará 500ms en abrir el Popover, pero solo 100ms para cerrarlo.
html Boolean false Especifica si se aceptan tags html en el Popover.
  • true: acepta tags HTML.
  • false: no se aceptan tags HTML.
Nota: el HTML debe ser insertado en el atributo title o usando la misma opción.
placement String "right" Especifica la posición del Popover. Posibles valores:
  • top: arriba
  • buttom: abajo
  • left: lado izquierdo
  • right: lado derecho
  • auto: El Navegador decide la posición, es decir, si el valor es "auto left" el Popover se visualizará a la izquierda mientras sea posible, de lo contrario,
    se mostrará a la derecha; si el valor es "auto bottom" el Popover se visualizará abajo mientras sea posible, de lo contrario se mostrará arriba.
selector String, Boolean false Agrega el Popover a un selector específico.
template String Se usa como base HTML para crear el Popover
El título del Popover debe ser inyectado dentro de .popover-title
El contenido del Popover debe ser inyectado dentro de .popover-content
.arrow se convertirá en la flecha del Popover
El elemento más externo es quien contendrá la clase .popover
title String "" Especifica el texto que se muestra en el título del Popover.
trigger String "click" Especifica como el Popover será activado. Posibles valores:
  • "click": activa el Popover con un click.
  • "hover": activa el Popover con el puntero del mouse.
  • "focus": activa el Popover cuando tiene el foco.
Nota: para pasar múltiples valores, se deben separar mediante un espacio.

Métodos

Se muestra la siguiente tabla de métodos disponibles:

Método Descripción
.popover(options) Activa el Popover con una opción. Ver la tabla de Opciones con los valores válidos.
.popover("show") Muestra el Popover.
.popover("hide") Oculta el Popover
.popover("toggle") Alterna el Popover.
.popover("destroy") Oculta y destruye el Popover.

Eventos

Se muestra la siguiente tabla de eventos disponibles:

Evento Descripción
show.bs.popover Se produce cuando el Popover está a punto de ser mostrado.
shown.bs.popover Ocurre cuando el Popover está totalmente muestrado (después de que la transición CSS es completada).
hide.bs.popover Se produce cuando el Popover está a punto de ser ocultado.
hidden.bs.popover Ocurre cuando el Popover está totalmente ocultado (después de que la transición CSS es completada).
