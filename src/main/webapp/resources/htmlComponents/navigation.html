Barras de navegación

Usando Bootstrap una barra de navegación se puede extender o colapsar dependiendo del tamaño de la pantalla.
Una barra de navegación estándar es creada usando: <nav class="navbar navbar-default">.


Ejemplo:


Barra de navegación invertida

Si no le gusta el estilo default de la barra de navegación, Bootstrap tiene una alternativa, barra de navegación de color negro, para esto solo cambie la clase .navbar-default por .navbar-inverse.



Barra de navegación fija

La barra de navegación además se puede fijar en la parte superior o inferior de la página. Una "fixed navigation bar" mantiene visible la barra en una posición (superior o inferior) independiente del scroll de la página.


La clase .navbar-fixed-top hace que la barra se fije en la parte superior de la página.

La clase .navbar-fixed-bottom hace que la barra se mantenga en la parte inferior.

Ejemplo:

			
				<nav class="navbar navbar-inverse navbar-fixed-top">
				  <div class="container-fluid">
				    <div class="navbar-header">
				      <a class="navbar-brand" href="#">Nombre del sitio</a>
				    </div>
				    <div>
				      <ul class="nav navbar-nav">
				        <li class="active"><a href="#">Home</a></li>
				        <li><a href="#">Pagina 1</a></li>
				        <li><a href="#">Pagina 2</a></li> 
				        <li><a href="#">Pagina 3</a></li> 
				      </ul>
				    </div>
				  </div>
				</nav>


Barra de navegación con Dropdown

La barra de navegación puede además contener menús estilo dropdown.



Barra de navegación a la derecha

La clase .navbar-right es usada para alinear a la derecha botones de la barra de navegación.
En el ejemplo del demo se puede ver como se inserta un botón "Sign Up" y un botón "Login" en la parte derecha de la barra navegación. Además podemos agregar un glyphicon para cada uno de los dos nuevos botones.



Barra de navegación colapsable

La barra de navegación ocupa mucho espacio en pantallas pequeñas. Podemos ocultar la barra de navegación y solo mostrarla cuando sea necesario. En la pestaña demo podemos ver el ejemplo la barra de navegación, está es reemplazada por un botón . El menu de navegación será desplegado solamente cuando se de click en el botón.
