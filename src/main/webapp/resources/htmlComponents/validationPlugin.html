Plugin de validación jQuery


Plugin de validación jQuery

Empezar a usarlo es tan simple como usar una línea de jQuery seleccionar una forma y aplicar el plugin de validación. $("#id_de_la_forma").validate();

Además hay que agregar unas cuantas anotaciones a cada elemento HTML para indicar las reglas de validación. Por supuesto esta no es la única forma para especificar reglas, no se tiene que confiar solo en los mensajes por default ya que estos se pueden personalizar

Para agregar el plugin a su página HTML solo agregue la librería jquery.validate.js o jquery.validate.min.js en el encabezado

		
			<!-- plugin de validación -->
			<script type="text/javascript" src="jquery.validate.min.js"></script>

Métodos del plugin

Esta librería agrega tres métodos del plugin jQuery, el punto de entrada principal va ser el metodo validate:

  • validate() - Valida la forma seleccionada.
  • valid() - Comprueba si la forma seleccionada o los elementos seleccionados son validos.
  • rules() - Lee, agrega y remueve reglas a un elemento.

Selectors personalizados

Esta librería además extiende jQuery con tres selectores personalizados:

  • :blank - Selecciona todos los elementos con valor en blanco.
  • :filled - Selecciona todos los elementos con un valor capturado.
  • :unchecked - Selecciona todos los elementos que no están comprobados.

Validator

El metodo validate retorna un objeto Validator que tiene algunos métodos públicos los puede usar para lanzar validación programáticamente o cambiar el contenido de la forma. El objeto validador tiene más métodos pero solamente los documentados aquí son destinados para el uso.

  • Validator.form() - Valida la forma.
  • Validator.element() - Valida un simple elemento.
  • Validator.resetForm() - Restaura la forma controlada.
  • Validator.showErrors() - Muestra los mensajes especificados.
  • Validator.numberOfInvalids() - Retorna el número de campos no validos.

Hay algunos métodos estáticos en el objeto validator:

  • jQuery.validator.addMethod() - Agrega un metodo personalizado de validación.
  • jQuery.validator.format() - Reemplaza {n} elementos con argumentos.
  • jQuery.validator.setDefaults() - Modifica las configuraciones por default para validación.
  • jQuery.validator.addClassRules() - Add a compound class method.

Lista de métodos de validación incluidos

Un conjunto de métodos estándar de validación están integrados:

  • required - Hace al elemento requerido.
  • remote - Solicita un recurso para hacer la validación del elemento.
  • minlength - Hace que el elemento acepte una mínima longitud.
  • maxlength - Hace que el elemento acepte una máxima longitud.
  • rangelength - Hace que el elemento acepte un rango de longitudes.
  • min - Hace que el elemento acepte un mínimo.
  • max - Hace que el elemento acepte un máximo.
  • range - Hace que el elemento acepte un rango de valores.
  • email - Hace que el elemento acepte como valor un email valido.
  • url - Hace que el elemento acepte como valor una url valida.
  • date - Hace que el elemento requiera una fecha.
  • dateISO - Hace que el elemento requiera una fecha ISO.
  • number - Hace que el elemento requiera un número decimal.
  • digits - Hace que el elemento acepte dígitos solamente.
  • creditcard - Hace que el elemento requiera un número de tarjeta de crédito.
  • equalTo - Requiere que el elemento sea igual a otro.
