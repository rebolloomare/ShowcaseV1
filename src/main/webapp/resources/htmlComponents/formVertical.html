Forma básica (Vertical)

Para construir una forma, necesitamos la etiqueta form con la clase form agregada:

Ejemplo:

 
		
	<form class="form">;
	</form>

Se recomienda usar los siguientes estandares:

  • Siempre usar <form role="form"> (ayuda en la accesibilidad para personas que usan pantallas lectoras)
  • Rodear a las etiquetas y controles de la forma entre <div class="form-group"> (necesarios para optimizar espacio)
  • Agregar la clase .form-control a todo los elementos de texto como <input>, <textarea>, y <select> con esto se ocupará el 100% de ancho
