Captcha Plugin

Este plugin está diseñado para ayudar a superar el envío automático de formularios al exigir una "persona real" para identificar el texto compuesto de puntos. El valor introducido se compara en el servidor con el valor generado para determinar si el proceso debe continuar. La funcionalidad del plugin real person se pueden añadir fácilmente a un campo de entrada con la configuración predeterminada correspondiente. También puede eliminar la funcionalidad real de la persona si ya no es necesario.


Uso

  1. Incluir la librería jQuery en la sección de cabecera de su página. 
    <script type="text/javascript" src="<c:url value='/resources/jquery/core/js/jquery.js'/>"></script>
  2. Descargar e incluir el CSS jQuery Real Person y JavaScript en la sección de cabecera de su página. 
    <link href="/resources/css/jquery.realperson.css'/" rel="stylesheet" type="text/css">
<script type="text/javascript" src="<c:url value='/resources/jquery/core/js/jquery.plugin.js'/>"></script>
<script type="text/javascript" src="<c:url value='/resources/jquery/core/js/jquery.realperson.js'/>"></script>
    Nota: alternativamente, puedes usar la versión minimizada jquery.realperson.min.js


    3. Configuración de instancia

    Personaliza cada campo de entrada con la configuración siguiente ( todos son opcionales ):

    $(selector).realperson({length: 4});

    chars | dot | dots | hashName | includeNumbers | length | regenerate

    Nombre   Tipo   Default   Comentario
    lengthnumber6 El número de caracteres a introducir.
    includeNumbersbooleanfalse Se establece en true para incluir los dígitos del 0 al 9 en los caracteres utilizados para generar la cadena de desafío. De lo contrario sólo se utilizan los caracteres alfabéticos.

    regeneratestring'Click to change' El texto que se mostrará para informar al usuario de que pueden hacer clic en el desafío de generar un nuevo valor - en caso de que no se puede leer fácilmente el valor original.
    hashNamestring'{n}Hash' El nombre del campo a utilizar para el valor hash que corresponde con el texto del desafío. Usar ' ( n }' en el entorno de sustituir el nombre del campo de entrada original
    dotstring'*' El carácter utilizado para generar los patrones de puntos de las letras de desafío.

    dotsstring[][]$.realperson.defaultDots Cada entrada en el nivel superior suministra los patrones de puntos para la posición correspondiente en chars. Cada entrada es una matriz de cadenas que proporcionan la línea de salto de línea abajo del patrón de puntos. Usar $.realperson.defaultDots para la configuración default.

    dots: ['   *   ', '  * *  ', '  * *  ', ' *   * ',
		' ***** ', '*     *', '*     *'], // A
	['****** ', '*     *', '*     *', '****** ',
		'*     *', '*     *', '****** '], // B
	...

    charsstring$.realperson.alphabetic La lista de caracteres para generar el reto. El patrón para cada carácter se toma de la posición correspondiente en dots. Puedes utilizar $.realperson.alphabetic ó $.realperson.alphanumeric para las secuencias de caracteres estandar.

    chars: 'ABCDEF'
chars: $.realperson.alphanumeric


    Funciones

    destroy | disable | enable | getHash | option (get) | option (set) | setDefaults

    Signature   Returns   Comments
    $.realperson.setDefaults(settings)RealPerson object Actualiza la configuración de instancia predeterminado para utilizar con todas las instancias de RealPerson.
    $(selector).realperson('option', options)jQuery object Actualiza la configuración de la instancia(s) persona real conectado a la entrada(s) dada.

    options (object) la colección de nuevas configuraciones.

    $(selector).realperson('option', {length: 8, includeNumbers: false});

    $(selector).realperson('option', name, value)jQuery object Actualización de una configuración particular de la instancia(s) de RealPerson conectadas a la entrada dada(s).

    name (string) el nombre de la configuración a cambiar;
    value (any) el nuevo valor de esa configuración.

    $(selector).realperson('option', 'length', 8);

    $(selector).realperson('option', name)object or any Recuperar una o todas las opciones actuales de la primera instancia RealPerson adjuntas a la entrada(s) dada.

    name (string, optional) el nombre de las configuraciones a recuperar; omitir para todos los ajustes.

    var settings = $(selector).realperson('option');
var length = $(selector).realperson('option', 'length');

    $(selector).realperson('enable')jQuery object Habilitar la funcionalidad de RealPerson para la entrada(s) dada.

    $(selector).realperson('disable')jQuery object Desactivar la funcionalidad de RealPerson para la entrada(s) dada.

    $(selector).realperson('getHash')number Recuperar el valor hash para el texto que se muestra actualmente .

    $(selector).realperson('destroy')jQuery object Retirar la funcionalidad de RealPerson desde la entrada(s) dada.


    Salt

    Haga su desafío más seguro, proporcionando un valor salt desde el servidor. Este valor se combina con el texto aleatorio para generar el valor hash.

    Puede suministrar el valor de salt mediante el establecimiento $.salt directamente antes de cargar el plugin RealPerson. El valor se elimina después de ser salvado por el plugin para evitar que otros vean el código (Sin embargo todavía sería visible en la fuente de la página).

    <script>$.salt = '123456';</script>
<script src="js/jquery.realperson.js"></script>

    Como alternativa, se puede suministrar el valor salt como el valor de un campo oculto con un ID de "salt" . El campo se borra después de haber sido salvado por el plugin para evitar que otros vean que en el código (Sin embargo todavía sería visible en la fuente de la página ) .

    <input type="hidden" id="salt" value="123456">

    También puede utilizar el $.salt valor para ajustar el selector para el campo oculto .

    <script>$.salt = '#mysalt';</script>
...
