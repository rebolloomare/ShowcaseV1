Máscaras



jQuery Mask Plugin

Su sintaxis es muy simple, primero, seleccione el input (o inputs necesarios) seguido por .mask y a continuación como parámetro se envia la máscara que se quiere aplicar al campo. jQuery Mask Plugin no tiene máscaras pre-establecidas, tu eliges la máscara como desees, ya que es posible definir otro tipo de datos en cada dígito de la máscara. Se puede enviar otro segundo párametro, el cual es totalmente opcional, como por ejemplo: Expresiones Regulares.


Implementación

Lo único que necesitas hacer es incluir el archivo javascript del plugin entre la etiqueta head en tu documento HTML y está todo listo para usar.
Función JavaScript
$(document).ready(function() {
			$('#fecha').mask('00/00/0000', {'translation': {0: {pattern: /[0-9*]/}}});
			$('#telefono').mask('0000-0000');
			$('#hora').mask('00:00:00');
			$('#fechayhora').mask('00/00/0000 00:00:00');
			$('#cp').mask('00000');
			$('#lada').mask('(00) 0000-0000');
			$('#pos').mask("00/00/0000", {placeholder: "__/__/____"});
		});
