Plugin Carousel

Es un componente de presentación de imágenes o diapositivas como un carrusel


Uso

Esta funcionalidad ya forma parte de Bootstrap, es decir, el codigo JavaScript y Hoja de Estilo se ecuentran en los archivos bootstrap.js y bootstrap.css respectivamente.

Clases

Clase Descripción
.carousel Crea un carrusel
.slide Agrega una transición CSS y efecto de animación cuando se desliza de un elemento al siguiente. Eliminar a esta clase si no desea este efecto.
.carousel-indicators Agrega indicadores para el carrusel. Estos son los pequeños puntos en la parte inferior de cada diapositiva (que indica el número de diapositivas que hay en el carrusel, y que se deslizan el usuario está viendo actualmente)
.carousel-inner Agrega diapositivas al carrusel
.item Especifica el contenido de cada diapositiva
.left carousel-control Agrega un botón izquierdo para el carrusel, que permite al usuario volver entre las diapositivas
.right carousel-control Agrega un botón a la derecha al carrusel, que permite al usuario ir hacia adelante entre las diapositivas
.carousel-caption Especifica una leyenda para el carrusel
