Botones

Usted puede crear fácilmente un botón con Bootstrap. Solo tiene que agregar la clase btn a uno de los elementos a convertir como a, button, o input para verse en un lindo botón de Bootstrap

Ejemplo:

	
	<a href="#" class="btn">Click Here</a>			
	<button type="button" class="btn">Click Here</button>
	<input type="button" class="btn  value="Click Here">


Estilos de botón

Bootstrap tiene siete tipos de estilos de botón:

  • .btn-default
  • .btn-primary
  • .btn-success
  • .btn-info
  • .btn-warning
  • .btn-danger
  • .btn-link

Solo agregue la clase de alguno de los estilos seleccionados



Tamaños de botón

Bootstrap tiene cuatro tamaños para botones:

  • .btn-lg
  • .btn-md
  • .btn-sm
  • .btn-xs

Solo agregue alguno de los estilos seleccionado.



Botones bloqueo de nivel:

Un botón bloqueo de nivel expande a todo lo ancho del elemento padre.
Agregar la clase .btn-block para crear un botón de bloqueo de nivel. 

	
	<button class="btn btn-primary btn-block" type="button">Button 1</button>


Botones Activos/Deshabilitados

Un botón puede configurarse como activo (presionado) o en estado deshabilitado (no se puede hacer click)
La clase .active hace que un botón aparezca presionado, y la clase .disabled hace a un botón deshabilitado: 

	
		<button class="btn btn-primary active" type="button">Activo</button>
		<button class="btn btn-primary disabled" type="button">Deshabilitado</button>
