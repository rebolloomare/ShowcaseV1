Generar Código de Barras

jQuery Barcode Plugin

El uso de este complemento es muy simple, solo se tiene que llamar a la función .barcode en el input, el cual toma como parametro el string de entrada para generar el código de barras. jQuery Barcode plugin soporta los siguientes tipos:
codabar
code11 (code 11)
code39 (code 39)
code93 (code 93)
code128 (code 128)
ean8 (ean 8)
ean13 (ean 13)
std25 (standard 2 of 5 - industrial 2 of 5)
int25 (interleaved 2 of 5)
msi
datamatrix (ASCII + extended)

Y puede generar código de barras en 4 tipos de salida:
CSS
BMP (not works in IE)
SVG (not works in IE)
CANVAS (sólo funciona con navegadores compatibles con canvas)


Implementación

Lo único que necesitas hacer es incluir el archivo javascript del plugin entre la etiqueta head en tu documento HTML y está todo listo para usar.
Función JavaScript
$(".btn").bind("click",function()
		               {
		                
		$("#barcode").barcode($("#txtbarcode").val(),"code128");
		               });
		$(".btn").click();
Invocación HTML
<div class="form-group">
			Texto : <input type="text" id="txtbarcode" value="iminfo.in"/>
			<br/><br/>
			<div id="barcode"></div>
			<br/><br/>
			<a class="btn btn-info">Generar Código de Barras</a>
	</div>
