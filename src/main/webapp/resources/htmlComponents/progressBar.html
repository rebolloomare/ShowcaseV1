Barra de Progreso/ProgressBar


Una barra de progreso se puede utilizar para mostrar al usuario qué tan avanzado está un proceso. Bootstrap ofrece varios tipos de barras de progreso.

Para crear una barra de progreso básica, se debe agregar la clase .progress al elemento <div>.
Por ejemplo:

<div class="progress">
<div class="progress-bar" role="progressbar" 
	aria-valuenow="70"  aria-valuemin="0" aria-valuemax="100" style="width:70%"></div>
</div>

Para agregar una etiqueta, basta agregar el texto con el signo de porcentaje (ver código Demo de este ejemplo).


Barras de Progreso Contextuales

Las clases contextuales son utilizadas para dar significado a través del color. Las clases contextuales que pueden usarse con las barras de progreso son:

  • .progress-bar-success
  • .progress-bar-info
  • .progress-bar-warning
  • .progress-bar-danger
En la pestaña Demo podrá ver ejemplos de barras de progreso con clases contextuales.

Barras de Progreso Tipo Cebra

Las barras de progreso tipo "Cebra" o con rayas (Striped) son un estilo característico de las barras de progreso. Para utilizar este estilo se debe agregar la clase .progress-bar-striped

En la pestaña Demo podrá ver ejemplos de barras de progreso con clases contextuales. 


Nota: Las barras de progreso no son compatibles con Internet Explorer 9 y versiones anteriores,
porque utilizan las transiciones y animaciones CSS3 para alcanzar algunos de sus efectos.
