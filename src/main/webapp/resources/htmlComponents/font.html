Font Awesome - Iconos para Bootstrap

Con Font Awesome puedes personalizar iconos instantáneamente - tamaño, color, sombra, y todo lo que se puede hacer con el poder de CSS.
Diseñado para ser totalmente compatible con Bootstrap.
Solo coloca los iconos en cualquier lugar con la etiqueta <i>.


Implementación

Lo único que necesitas hacer es incluir el archivo .css entre la etiqueta head en tu documento HTML y está todo listo para usar.
