Control de estados en formas Bootstrap

Entrada de texto

Bootstrap soporta todos los tipos de entrada de texto HTML5: text, password, datetime, datetime-local, date, month, time, week, number, email, url, search, tel, y color.

Campos estáticos

Si usted necesita insertar texto estático al lado de una etiqueta de formulario en una forma horizontal, use la clase .form-control-static en una etiqueta <p>

Control de estados en formas Bootstrap

  • INPUT FOCUS - El fuera de linea para este campo es removido y una caja sombreada es aplicada en foco
  • DISABLED INPUTS - Agrega el atributo disabled para deshabilitar la entrada al campo.
  • DISABLED FIELDSETS - Agrega el atributo disabled a un <fieldset> para deshabilitar a todos los campos dentro de el.
  • READONLY INPUTS - Agrega el atributo readonly a un campo para pevenir la entrada de datos.
  • VALIDATION STATES - Bootstrap incluye estilos para validaciones para error, warning y mensajes de exito. Para usarlos agregue alguna de las clases .has-warning, .has-error, o .has-success a un elemento padre.
  • ICONS - Usted puede agregar iconos de retroalimentación con la clase .has-feedback y un icono
  • HIDDEN LABELS - Agregue una clase .sr-only en una etiqueta no visible
