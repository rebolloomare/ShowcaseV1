Widget de Slider


El widget Slider de jQuery UI muestra elementos en forma de deslizadores. Hay varias opciones y rangos. El indicador se puede mover con el mouse o las teclas de flecha.

ui-slider: La clase principal del control Slider. Este elemento tiene las clases adicionales de ui-slider-horizontal o ui-slider-vertical dependiendo de la orientación (orientation) del slider.
ui-slider-range: Esta clase es utilizada cuando se desea establecer un rango (range) en el valor. Este elemento tiene las clases adicionales de ui-slider-range-min o ui-slider-range-max si la opción de range se desea establecer como "min" o "max" respectivamente.

Opciones

animate

Para deslizar el control deslizante suavemente cuando el usuario hace clic en la guía de desplazamiento.
Tipos: Boolean, String , Number.
Default: false

Múltiples valores soportados:
Boolean: Con valor true, el control deslizante será animado con la duración predeterminada.
String: La velocidad, como "fast" o "slow".
Number: La duración de la animación, en milisegundos.
Ejemplos de Código
Inicializando el Slider con la opción animate definida:
$( ".selector" ).slider({
animate: "fast"
});
Obtener o establecer la opción animate, después de inicializar:
// Getter
var animate = $( ".selector" ).slider( "option", "animate" );
// Setter
$( ".selector" ).slider( "option", "animate", "fast" );

disabled

Tipo: Boolean.
Default: false
Ejemplos de Código
Inicializando el Slider con la opción disabled definida:
$( ".selector" ).slider({
disabled: true

Obtener o establecer la opción disabled, después de inicializar:
// Getter
var disabled = $( ".selector" ).slider( "option", "disabled" );
// Setter

min

El mínimo valor del Slider.
Tipo: Number.
Default: 0
Ejemplos de Código
Inicializando el Slider con la opción min definida:

min: 10

Obtener o establecer la opción min, después de inicializar:

orientation

Determina si los controles deslizantes se mueven horizontalmente (a la izquierda el valor min, max a la derecha) o verticalmente (valor min en la parte inferior, el max en la parte superior).
Posibles valores: : "horizontal", "vertical".
Tipo: String.
Default: "horizontal"
Ejemplos de Código
Inicializando el Slider con la opción orientation definida:

orientation: "vertical"
});
Obtener o establecer la opción orientation, después de inicializar:
// Getter
var orientation = $( ".selector" ).slider( "option", "orientation" );
// Setter
$( ".selector" ).slider( "option", "orientation", "vertical" );

range

Usado cuando el Slider requiere representar un rango.
Tipos: Boolean y String.
Default: false

Múltiples valores soportados:
Boolean: Con valor true, el Slider detecta los dos controles deslizantes y crea un elemento de rango entre las dos asas.
String: Puede ser "min" o "max". El rango min va desde el valor mínimo del Slider a el control deslizante. El rango max va desde el control deslizante a el valor máximo del Slider.
Ejemplos de Código
Inicializando el Slider con la opción animate definida:

range: true
});
Obtener o establecer la opción animate, después de inicializar:
// Getter
var range = $( ".selector" ).slider( "option", "range" );
// Setter
$( ".selector" ).slider( "option", "range", true );

step

Determina el tamaño o la cantidad de cada intervalo que tarda el control deslizante entre los valores mínimo y máximo. El rango de valores completo de la corredera determinado (max - min) debe ser divisible por el valor establecido en step.
Tipo: Number.
Default: 1
Ejemplos de Código
Inicializando el Slider con la opción step definida:

});
Obtener o establecer la opción step, después de inicializar:

// Setter
$( ".selector" ).slider( "option", "step", 5 );

value

Determina el valor del deslizador, si sólo hay un control deslizante. Si hay más de un control, determina el valor de la primera asa.
Tipo: Number.
Default: 0
Ejemplos de Código
Inicializando el Slider con la opción value definida:

// Setter

values

Esta opción se puede utilizar para especificar varios controles. Si la opción de rango se establece en true, la longitud de los valores debe ser de 2.
Tipo: Array
Default: null
Ejemplos de Código
Inicializando el Slider con la opción value definida:

Obtener o establecer la opción value, después de inicializar:
// Getter

// Setter

Métodos

Método Descripción Ejemplo
destroy()
 Elimina la funcionalidad por completo. Esto devolverá el elemento de nuevo a su estado previo de inicio.
Este método no acepta ningún argumento.
$( ".selector" ).slider( "destroy" );
disable()
 Inhabilita el Slider.
Este método no acepta ningún argumento.
$( ".selector" ).slider( "disable" );
enable()
 Habilita el Slider.
Este método no acepta ningún argumento.
$( ".selector" ).slider( "enable" );
instance()
 Recupera el objeto de instancia del control deslizante. Si el elemento no tiene una instancia asociada,
devuelve undefined. A diferencia de otros métodos de controles, instance() es seguro
llamar en cualquier elemento después de que el plugin de control deslizante se ha cargado.
Este método no acepta ningún argumento.
$( ".selector" ).slider( "instance" );
option(optionName)
 Obtiene el valor especificado en optionName.
Para las opciones que tienen objetos como su valor, se puede obtener el valor de una clave específica
mediante el uso de la notación de punto. Por ejemplo, "foo.bar" obtendría el valor de
la propiedad de barra en la opción foo.
optionName: el nombre de la opción a obtener.
Type: String
var isDisabled = $( ".selector" ).slider( "option", "disabled" );
option()
 Recupera un objeto que contiene los pares de atributo/valor que representan las actuales opciones del Slider.
Este método no acepta ningún argumento.
var options = $( ".selector" ).slider( "option" );
option(optionName, value)
 Establece el valor de la opción del Slider especificada en optionName.
Para las opciones que tienen objetos como su valor, se puede establecer el valor de una propiedad
específica mediante el uso de la notación de punto. Por ejemplo, "foo.bar" actualiza
el valor de la propiedad de barra en la opción foo.
optionName: el nombre de la opción a obtener.
Type: String
value: el valor a establecer en la opción.
Type: Object
$( ".selector" ).slider( "option", "disabled", true );
option(options)
 Establece uno o más opciones al Slider.
options: un mapa de pares de opciones-valores a establecer.
Type: Object
$( ".selector" ).slider( "option", { disabled: true } );
value()
 Obtiene el valor del Slider.
Este método no acepta ningún argumento.
var selection = $( ".selector" ).slider( "value" );
value(value)
 Establece el valor al Slider.
value: el valor a establecer.
Type: Number
$( ".selector" ).slider( "value", 55 );
values()
 Obtiene el valor de todos los controles deslizantes del Slider.
Este método no acepta ningún argumento.
var selection = $( ".selector" ).slider( "value" );
values(index)
 Obtiene el valor para un control especificado.
index: el índice del control.
Type: Integer
var value = $( ".selector" ).slider( "values", 0 );
values(index, value)
 Establece el valor para un control especificado.
index: el índice del control.
Type: Integer
value: el valor a establecer.
Type: Number
$( ".selector" ).slider( "values", 0, 55 );
values(values)
 Establece el valor para todos los controles.
index: el índice del control.
Type: Integer
values: los valores a establecer.
Type: Array
$( ".selector" ).slider( "values", [ 55, 105 ] );
widget()
 Regresa un objeto jQuery conteniendo el Slider.
Este método no acepta ningún argumento.
var widget = $( ".selector" ).slider( "widget" );

Eventos

change(event, ui)

Se activa cuando el usuario desliza un control deslizante, si el valor ha cambiado; o si el valor se cambia mediante código a través del método value.

  • event

    Tipo: Event

  • ui

    Tipo: Object

    • handle

      Tipo: jQuery
      El objeto jQuery que representa el mango que se ha cambiado.

    • value

      Tipo: Number
      El valor actual del Slider.

Ejemplo. Inicializando el Slider con la llamada al evento change:
$( ".selector" ).slider({
change: function( event, ui ) {}
});

create(event, ui)

Se activa cuando el Slider es creado.

  • event

    Tipo: Event

  • ui

    El objeto ui es vacío pero es incluido para consistencia con los otros eventos.

Ejemplo. Inicializando el Slider con la llamada al evento create:
$( ".selector" ).slider({
create: function( event, ui ) {}
});

slide(event, ui)

Se activa en cada movimiento del mouse durante el deslizado. El valor proporcionado en el evento como ui.value representa el valor que el mango tendrá como resultado del movimiento actual. Cancelar el evento evitará que el mango se mueva y el mango seguirá teniendo su valor anterior.

  • event

    Tipo: Event

  • ui

    Tipo: Object

    • handle

      Tipo: jQuery
      El objeto jQuery que representa el mango que se ha movido.

    • value

      Tipo: Number
      El valor actual del Slider.

    • values

      Tipo: Array
      Un arreglo de valores para un Slider de tipo rango.

Ejemplo. Inicializando el Slider con la llamada al evento slide:
$( ".selector" ).slider({
slide: function( event, ui ) {}
});

start(event, ui)

Se activa cuando el usuario comienza el deslizado.

  • event

    Tipo: Event

  • ui

    Tipo: Object

    • handle

      Tipo: jQuery
      El objeto jQuery que representa el mango que se ha movido.

    • value

      Tipo: Number
      El valor actual del Slider.

Ejemplo. Inicializando el Slider con la llamada al evento start:
$( ".selector" ).slider({
start: function( event, ui ) {}
});

stop(event, ui)

Se activa después de que el usuario ha movido el control.

  • event

    Tipo: Event

  • ui

    Tipo: Object

    • handle

      Tipo: jQuery
      El objeto jQuery que representa el mango que ha sido movido.

    • value

      Tipo: Number
      El valor actual del Slider.

Ejemplo. Inicializando el Slider con la llamada al evento stop:
$( ".selector" ).slider({
stop: function( event, ui ) {}
});
